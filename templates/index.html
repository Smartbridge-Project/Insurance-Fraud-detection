About FraudShield Pro

A complete end-to-end machine learning pipeline for auto insurance fraud detection — built on the Oracle Insurance Fraud dataset with 15,420 records, covering data preparation, EDA, model building, evaluation, and Flask deployment.

🗂 Dataset

The Oracle Insurance Fraud dataset contains 15,420 auto insurance claims with 33 features covering incident details, policyholder demographics, vehicle info, and claim history.


Source: fraud_oracle.csv
Records: 15,420 rows × 33 columns
Fraud rate: 6.0% (923 cases)
Missing values: None

⚙️ ML Pipeline

1
Data Collection

Loaded fraud_oracle.csv, explored shape & distributions

2
Data Preparation

Dropped irrelevant columns (PolicyNumber, RepNumber), no nulls to handle

3
EDA & Visualization

Univariate, bivariate & multivariate analysis with charts

4
Encoding & Scaling

LabelEncoder for 24 categorical cols, StandardScaler on all features

5
Model Training

6 classifiers trained on 80/20 stratified split

📋 Models Used

Six classification algorithms were trained, compared, and evaluated:


🌳 Decision Tree (max_depth=5) — 94.13%
🌲 Random Forest (100 estimators) — 94.07%
📈 Logistic Regression (max_iter=500) — 94.00%
📍 K-Nearest Neighbors (k=7) — 94.00%
⚡ Support Vector Machine (RBF) — 94.00%
📊 Naive Bayes (Gaussian) — 84.95%

🚀 Deployment

The best model is saved with joblib and served via a Flask web application with a 4-page UI: Home, Predict, Analysis, and About.


Framework: Flask (Python)
Frontend: HTML5, CSS3, Vanilla JS
Serialization: joblib
Preprocessing: scikit-learn


Run with: python app.py
Then open: http://localhost:5000
